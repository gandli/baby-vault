# BabyVault 👶

> **Baby growth records that stay at home — privacy-first, AI-powered.**

## Why BabyVault?

🤱 **One-handed operation** — All core actions reachable by thumb while holding your baby

🏠 **Data stays home** — Photos and videos sync to your home server (Mac Mini / NAS), never to the cloud

🧒 **AI play coach** — Daily age-specific activity recommendations based on your baby's milestones

## Core Features

- 📸 **Growth Timeline** — Photos/videos auto-archived by baby's age with AI tagging
- 🏆 **Milestone Tracker** — 4 dimensions (gross motor / fine motor / language / social)
- 🎮 **Daily Play Recs** — Age-appropriate parent-child activities
- 🍼 **Feeding Diary** — New food tracking with allergy alerts
- 📊 **Monthly Report** — AI-generated growth summary

## Tech Stack

Flutter · SQLite · WebDAV · Core ML · Bonjour

## Philosophy

> *"Baby's first steps shouldn't live on someone else's server."*
